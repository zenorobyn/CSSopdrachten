Welke verschillen zie je?

Ik zie vooral bij het linkerscherm dat van vives is. Dat ze meer gebruik maken van html code en in tabellen gaan werken.

Ik zie vooral bij het rechterscherm dat van vrt. Dat er meer gebruik maken van css code en dat ze toch anders gaan werken. Hierbij maken ze gebruik van van meer links en verwijzingen ergens naartoe.
